О проекте

Основная команда разработки это delphi/sql/web разработчики. Основной программный продукт это самописная ERP на delphi + ms sql. Планируем передать часть backend процессов python модулям – интеграции, манипулирование данными. Также в delphi есть возможность интеграции python (python4delphi), хочется посмотреть что из этого может выйти. Вы будете первым python разработчиком, так что вокруг вас перспективы организации команды.

Обязанности:

  • Разработка новых и оптимизация существующих программных модулей;

  • Тестирование и отладка разработанного кода, ведение документации;

  • Участие в наборе и организации команды python разработчиков.

Требования:
  • Опыт разработки на python 5+ лет. Опыт написания асинхронных и многопоточных приложений. Желателен опыт разработки GUI под Windows;
  • Отличное знание SQL(t-sql), лучше именно MS SQL;
  • Умение писать простой и понятный код;
  • Активность и работоспособность.
Условия:
  • Оформление по ТК РФ;
  • График работы: с 08.30 до 17.00 (в пт до 16.30), сб и вс - выходные. (возможно изменение графика по желанию);
  • Своевременная оплата труда;
  • Соц пакет после года работы (ДМС, фитнес - на выбор, при работе в офисе);
  • Молодой и дружный коллектив. Корпоративные праздники;
  • Бесплатное питание (при работе в офисе).
